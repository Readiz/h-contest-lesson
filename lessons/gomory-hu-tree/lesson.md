# Gomory-Hu Tree

Gomory-Hu Tree는 독립 레슨 카드가 아니라 [Graph Cut Structures](../graph-cut-structures/lesson.md) 허브의 하위 페이지로 이동했습니다.

- 새 위치: [Graph Cut Structures / Gomory-Hu Tree](../graph-cut-structures/pages/gomory-hu-tree.md)
- trace page: [Gomory-Hu Construction Trace](../graph-cut-structures/pages/gomory-hu-tree-construct-trace.md)

이 이동 문서는 기존 링크를 따라온 독자를 안내하기 위해 남깁니다. manifest의 공개 레슨 목록에서는 `graph-cut-structures` 허브만 노출합니다.
