# Palindrome Query Structures

Palindrome Query Structures는 독립 레슨 카드가 아니라 [Palindrome Structures](../palindrome-structures/lesson.md) 허브의 하위 페이지로 이동했습니다.

- 새 위치: [Palindrome Structures / Palindrome Query Structures](../palindrome-structures/pages/palindrome-query-structures.md)
- 함께 읽을 페이지: [Palindrome Model Map](../palindrome-structures/pages/palindrome-model-map.md), [Palindrome Range DP](../palindrome-structures/pages/palindrome-range-dp.md)

이 이동 문서는 기존 링크를 따라온 독자를 안내하기 위해 남깁니다. manifest의 공개 레슨 목록에서는 `palindrome-structures` 허브만 노출합니다.
