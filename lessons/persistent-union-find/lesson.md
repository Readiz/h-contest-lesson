# Persistent Union-Find

Persistent Union-Find는 독립 레슨 카드가 아니라 [Versioned Data Structures](../versioned-data-structures/lesson.md) 허브의 하위 페이지로 이동했습니다.

- 새 위치: [Versioned Data Structures / Persistent Union-Find](../versioned-data-structures/pages/persistent-union-find.md)
- 함께 읽을 페이지: [Versioning Model Map](../versioned-data-structures/pages/versioning-model-map.md), [Persistent Queue and Stack](../versioned-data-structures/pages/persistent-queue-stack.md)

이 이동 문서는 기존 링크를 따라온 독자를 안내하기 위해 남깁니다. manifest의 공개 레슨 목록에서는 `versioned-data-structures` 허브만 노출합니다.
