# Suffix Automaton Applications

Suffix Automaton Applications는 독립 레슨 카드가 아니라 [Suffix and Periodicity Structures](../suffix-periodicity-structures/lesson.md) 허브의 하위 페이지로 이동했습니다.

- 새 위치: [Suffix and Periodicity Structures / Suffix Automaton Applications](../suffix-periodicity-structures/pages/suffix-automaton-applications.md)
- 함께 읽을 페이지: [Suffix Automaton](../suffix-periodicity-structures/pages/suffix-automaton.md), [Generalized Suffix Automaton](../suffix-periodicity-structures/pages/generalized-suffix-automaton.md)

이 이동 문서는 기존 링크를 따라온 독자를 안내하기 위해 남깁니다. manifest의 공개 레슨 목록에서는 `suffix-periodicity-structures` 허브만 노출합니다.
