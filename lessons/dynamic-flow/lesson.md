# Dynamic Flow

Dynamic Flow는 독립 레슨 카드가 아니라 Dynamic Network Optimization 허브의 하위 페이지로 이동했습니다.

- 새 위치: [Dynamic Network Optimization / Dynamic Flow](../dynamic-network-optimization/pages/dynamic-flow.md)
- 함께 읽을 페이지: [Dynamic Network Optimization / Practice Set](../dynamic-network-optimization/pages/practice-set.md)

이 이동 문서는 기존 링크를 따라온 독자를 안내하기 위해 남깁니다. manifest의 공개 레슨 목록에서는 `dynamic-network-optimization` 허브만 노출합니다.
